# Dataset

This project uses the **CIFAR-10** dataset (binary batch files).

⚠️ The full dataset is not uploaded to GitHub because it exceeds GitHub’s 25MB upload limit.

## How to access the dataset (Kaggle)
- Kaggle path used in this project:
  `/kaggle/input/cifar-10-image/cifar-10-batches-py`

## Files used (from Kaggle input)
- data_batch_1 ... data_batch_5
- test_batch
- batches.meta
